## NEW CATEGORY: Classical Conditionals (若/苟/則)

### cc31 — 若/如 conditional (philosophical parable register)

**Source:** 《莊子》· 逍遙遊  
**Text:** 且夫水之積也不厚，則其負大舟也無力。  
**Pinyin:** Qiě fú shuǐ zhī jī yě bù hòu, zé qí fù dà zhōu yě wú lì.  
**Modern Chinese:** 再说如果水积蓄得不够深，那么它就没有力量承载大船。  
**English:** Moreover, if the water's accumulation is not deep enough, then it lacks the strength to bear a great boat.  

**AST Structure Notes:**
- **Root:** Parallel Sentence with two clauses linked by 則 (consequence marker)
- **Clause 1 (Condition):** Topic = 水之積也 (nominalized phrase: "the water's accumulation"), Comment = 不厚 (negated adjectival predicate)
  - 之 functions as genitive marker inside the nominalization
  - 也 marks the topic pause
- **Clause 2 (Consequence):** Adjunct(則, conditional consequence) → Topic = 其 (anaphoric pronoun referring back to 水), Comment = VP(負大舟也無力)
  - 其 creates a co-reference arc back to the first clause's topic — the tree makes this visible
- **Key structural feature:** The 則 is an Adjunct (consequence marker), NOT a conjunction — it modifies the second clause's Comment, not the sentence root. This is the critical teaching point: 則 doesn't "join" clauses the way English "then" does; it marks the second clause as the result scope.

**Cross-dataset echo:** Modern Conditionals (如果…就…) — 則 parallels 就

### cc32 — 苟 conditional (political rhetoric register)

**Source:** 《戰國策》· 齊策  
**Text:** 苟無民，何以有君？  
**Pinyin:** Gǒu wú mín, hé yǐ yǒu jūn?  
**Modern Chinese:** 如果没有人民，怎么会有君主呢？  
**English:** If there were no people, how could there be a ruler?  

**AST Structure Notes:**
- **Root:** Sentence with two clauses — conditional + rhetorical question
- **Clause 1 (Condition):** Adjunct(苟, conditional marker) → Comment = 無民 (negated existential: "no people")
  - Topic is null — 苟 opens a hypothetical frame without needing an anchor
- **Clause 2 (Rhetorical consequence):** Fronted interrogative object 何 → Coverb phrase 以(何) → VP = 有君
  - **何以** is a classic object fronting: 何 (what) is the fronted object of 以 (by means of). Modern word order would be 以何.
  - The tree shows the displacement explicitly: Object(何) precedes its governing Preposition(以)
- **Key structural feature:** This sentence is a double showcase — conditional frame (苟) PLUS object fronting (何以). The tree reveals both simultaneously. This is exceptionally high pedagogical value.

**Cross-dataset echo:** Modern Conditionals + modern 用什么 (instrumental question)

## NEW CATEGORY: Classical Causatives (使/令)

### cc33 — 使-causative (military/political register)

**Source:** 《戰國策》· 趙策  
**Text:** 秦王使人謂安陵君曰。  
**Pinyin:** Qín wáng shǐ rén wèi Ānlíng jūn yuē.  
**Modern Chinese:** 秦王派人对安陵君说。  
**English:** The King of Qin sent someone to say to Lord Anling.  

**AST Structure Notes:**
- **Root:** Sentence → Topic(秦王, Agent) + Comment
- **Comment:** Head Verb(使, causative) → Embedded Clause
  - **Pivot:** Subject(人, Agent of embedded clause) — this is the structural pivot shared between 使 and the embedded VP
  - **Embedded VP:** Serial verb chain: 謂(say to) + Object(安陵君) + 曰(say/quote)
- **Key structural feature:** Classic pivotal construction — 人 is simultaneously the object of 使 (the person being sent) and the subject of 謂…曰 (the person doing the saying). The tree exposes this dual role through the Embedded Clause wrapper, identical to modern 让/叫 pivotals.
- **Note:** 使 here retains full lexical verb force ("dispatch/send") — compare with its grammaticalized descendant 使 in modern formal Chinese where it means "cause to."

**Cross-dataset echo:** Modern Pivotal Constructions (兼語句) — 使 parallels 让/派

### cc34 — 令-causative (Tang prose register)

**Source:** 韓愈《師說》  
**Text:** 惑而不從師，其為惑也終不解矣。  
**Pinyin:** Huò ér bù cóng shī, qí wéi huò yě zhōng bù jiě yǐ.  
**Modern Chinese:** 有了疑惑却不跟从老师（学习），那些疑惑最终不会解开了。  
**English:** If one is confused but does not follow a teacher, those confusions will ultimately never be resolved.  

**AST Structure Notes:**
- **Root:** Parallel Sentence — Condition clause + Consequence clause
- **Clause 1 (Condition):** Topic = [implicit generic person, ghost node], Comment contains two parallel VPs linked by 而(contrastive connective):
  - VP1: 惑 (be confused — stative verb as predicate)
  - Adjunct(而, contrastive: "but") → VP2: 不從師 (not follow a teacher)
- **Clause 2 (Consequence):** Topic = 其為惑也 (those confusions), Comment = 終不解矣
  - 其 = anaphoric pronoun (cross-referencing 惑 from clause 1)
  - 為惑 = copular predication ("being confusion")
  - 終 = Adjunct(time: "ultimately")
  - 不解 = negated VP
  - 矣 = Particle(assertion/finality)
- **Key structural feature:** This sentence demonstrates the causative *implication* without an explicit 令/使 marker — the logical structure is: "NOT following a teacher → confusion persists." The tree reveals the implicit causative logic through the parallel clause structure. It also showcases 而 in contrastive mode, reinforcing the existing Sequential Actions category.

**Cross-dataset echo:** Modern Pivotal Constructions + Sequential Actions (而)

## EXPAND: Object Fronting (賓語前置)

### cc35 — 之-marked object fronting (affirmative context)

**Source:** 《莊子》· 養生主  
**Text:** 吾生也有涯，而知也無涯。  
**Pinyin:** Wú shēng yě yǒu yá, ér zhī yě wú yá.  
**Modern Chinese:** 我的生命是有限的，但知识是无限的。  
**English:** My life has limits, but knowledge has no limits.  

**AST Structure Notes:**
- **Root:** Parallel Sentence — two antithetical clauses linked by 而(contrastive)
- **Clause 1:** Topic = 吾生也 (my life + topic pause), Comment = 有涯 (has limits)
  - 吾 = possessive modifier (classical first person)
  - 也 = topic pause marker
- **Clause 2:** Adjunct(而, contrastive) → Topic = 知也 (knowledge), Comment = 無涯 (has no limits)
- **Key structural feature:** The antithetical parallelism — 有涯 vs 無涯 — creates a mirror-image tree structure. Both clauses share identical Comment architecture (existential verb + object) but with opposing polarity. The tree reveals this symmetry beautifully.
- **Note:** While this sentence doesn't showcase canonical object fronting (賓語前置), it demonstrates the **也-marked topic** pattern that is the precondition for understanding fronting. Consider this a scaffolding example.

### cc35 (revised) — 之-marked object fronting (affirmative context)

**Source:** 《莊子》· 秋水  
**Text:** 牛馬四足，是謂天。  
**Pinyin:** Niú mǎ sì zú, shì wèi tiān.  
**Modern Chinese:** 牛马有四只脚，这叫做天然。  
**English:** That cattle and horses have four feet — this is called "natural."  

**AST Structure Notes:**
- **Root:** Sentence → Topic(牛馬四足, nominalized fact) + Comment(是謂天)
- **Comment:** 是 functions as an anaphoric pronoun ("this") resuming the Topic — fronted before the verb 謂(call).
  - 是 = Fronted Object (referring to the entire Topic proposition)
  - 謂 = Head Verb ("call/name")
  - 天 = Complement ("natural/heaven-given")
- **Key structural feature:** 是 here is NOT the modern copula but a **demonstrative pronoun** functioning as a fronted object. The tree shows 是 occupying the pre-verbal Object slot, with an arc back to the Topic clause it resumes. This is the most pedagogically important point: learners must distinguish classical 是(this) from modern 是(is).

**Cross-dataset echo:** Modern 是…的 constructions (是 as focus marker vs. classical 是 as demonstrative)

### cc36 — Interrogative fronting with 何/安

**Source:** 《史記》· 項羽本紀  
**Text:** 大丈夫不能自食，安能食人？  
**Pinyin:** Dà zhàng fū bù néng zì shí, ān néng shí rén?  
**Modern Chinese:** 大丈夫不能养活自己，怎么能养活别人？  
**English:** If a great man cannot feed himself, how can he feed others?  

**AST Structure Notes:**
- **Root:** Parallel Sentence — thesis clause + rhetorical question
- **Clause 1 (Thesis):** Topic = 大丈夫 (great man), Comment = 不能自食
  - 自 = reflexive Adjunct ("self")
  - 食 = Head Verb ("feed/eat")
- **Clause 2 (Rhetorical question):** Fronted interrogative 安(how/where) → 能食人
  - **安** = Fronted Adjunct (interrogative manner adverb). In modern Chinese this would be post-Topic: 怎么能…. In classical, it fronts to clause-initial position.
  - 食人 = VP (feed others)
- **Key structural feature:** The tree shows 安 in the fronted Adjunct position — displaced from its canonical post-Topic slot. The parallel structure with clause 1 (which has the adverb 不 in normal position) makes the displacement visible by contrast.

**Cross-dataset echo:** Modern Rhetorical Patterns (反問) + Object Fronting

## EXPAND: Nominalizers (者/所)

### cc37 — 之所以 construction ("the reason why")

**Source:** 《莊子》· 秋水  
**Text:** 井蛙不可以語於海者，拘於虛也。  
**Pinyin:** Jǐng wā bù kě yǐ yǔ yú hǎi zhě, jū yú xū yě.  
**Modern Chinese:** 井底之蛙不能和它谈论大海，是因为它被狭小的空间所局限。  
**English:** You cannot discuss the sea with a well-frog — it is confined by its narrow space.  

**AST Structure Notes:**
- **Root:** Sentence → Topic(井蛙不可以語於海者) + Comment(拘於虛也)
- **Topic:** Full clause nominalized by 者 — "the (reason/fact that) one cannot discuss the sea with a well-frog"
  - 井蛙 = Subject
  - 不可以 = negated potential
  - 語於海 = VP with coverb phrase (語 = speak, 於海 = about the sea)
  - 者 = nominalizer, converting the entire clause into a noun phrase
- **Comment:** 拘於虛也 = "is confined by narrowness"
  - 拘 = Head Verb (confined)
  - 於虛 = Coverb phrase (by/in narrowness)
  - 也 = assertion particle
- **Key structural feature:** The 者 nominalizer wraps an entire complex clause (with its own subject, negation, potential marker, and coverb phrase) into a single Topic node. The tree reveals the massive depth hidden inside what appears to be a simple two-part sentence. This is the FLS Matryoshka Principle in its purest classical form.

**Cross-dataset echo:** Modern Nominalizers + Coverbs & Prepositions (於/以/焉)

### cc38 — 者 in conditional/generic sense ("whoever…")

**Source:** 《禮記》· 大學  
**Text:** 欲治其國者，先齊其家。  
**Pinyin:** Yù zhì qí guó zhě, xiān qí qí jiā.  
**Modern Chinese:** 想要治理好国家的人，要先管理好自己的家。  
**English:** Whoever wishes to govern their state must first regulate their family.  

**AST Structure Notes:**
- **Root:** Sentence → Topic(欲治其國者) + Comment(先齊其家)
- **Topic:** Clause nominalized by 者 in conditional/generic sense
  - 欲 = Adjunct (desire/wish — modal)
  - 治 = Head Verb (govern)
  - 其國 = Object (their state)
  - 者 = nominalizer, but here meaning "whoever (does X)" — conditional/generic reading
- **Comment:** Adjunct(先, sequence: "first") → VP(齊其家)
  - 齊 = Head Verb (regulate/order)
  - 其家 = Object (their family)
- **Key structural feature:** 者 here does NOT mean "the person who" (definite) — it means "whoever/if one" (generic conditional). This is a different semantic function from cc2/cc12/cc23. The tree looks structurally identical (者-nominalized Topic → Comment), but the `subRole` should be marked as `conditional nominalizer` to distinguish it from the definitional usage. This teaches learners that 者 has multiple functions visible only through context, not tree shape.

**Cross-dataset echo:** Modern Nominalizers + the implicit conditional logic mirrors modern 如果…就… (without the explicit markers)
